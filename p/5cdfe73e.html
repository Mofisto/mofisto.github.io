无聊时的更新

发表于 2020-12-07 更新于 2021-03-02 分类于 笔记 阅读次数： Valine：
本文字数： 2.6k
这是一个我日常更新自己近期生活状态、看书笔记的页面。

今日杭下雨，冷的不想起床，躺尸了一上午，下午还是起来跑来图书馆看东西。争取今天把 linux 0.12 修改编译成功。

发现自己已经养成习惯，在家里看不进去东西。

专注于一个问题。为了解决一个问题而引入新的问题。说明解决问题的思路不正确。

今日上午继续更改 linux 0.12 源码，下午开始看师傅的直播。——2020.12.19

看到有留言说，以前的文章没有了。原因有几个：

  • 之前的域名时备案的，为了个人信息，就取消的备案；
  • 备案取消导致了一系列的连锁反应，和这个域名有关的链接全部无法访问，所以以前的图片无法访问，但是我本地有图片，所以就需要迁移，迁移又非常麻烦（个人懒得搞）；
  • 之前电脑 BitLocker 加密，微软账户又没有同步密钥，算是过去的我勒索了现在的我，很遗憾，全部数据丢失（包括原博客的一部分原始文件）;
  • 之前的文章，现在看，个人都觉得比较垃圾，如果真的有人想看，这里还有一份镜像。暂时我还不会删除，目前的源是在 github 上，所以速度可以有点慢，后面文章多了，可能考虑迁移到 gitee ,用来加速访问。;
  • 吸取前面的教训，这次的数据都放在私人远程库中，这时候真的发现 git 版本控制用来做什么都很方便。

今日继续改 linux 0.12 的源码。

我觉得我应该歇一下，短暂的放弃下，改代码真的太困难了。

应该专注于问题本身，而不是一直折腾环境（关键我一直折腾不出来)，这对我没有意义。

————2020.12.20

今天脑抽突然想到了两个人，两个女孩。

第一个女孩的故事是这样，某年我拿到了一张演唱会的票，但是需要我帮忙售卖周边，其实可以不用，因为但是给出的票太多，人非常多。我想了想，还是去帮忙了，于是认识了一个一起来帮忙的一个女孩。其实演唱会就是翻车现场，我是在受不了耳朵的折磨，陪着那位去卖周边，当时杭下雪，路面结冰。
结束之后，我问她，你往那边走？她说：”我往那边”。
我看了看，和我相反的方向，她没有伞，我也没送她伞。
于是，我说：“我走，这边”。
回去的匆匆赶上地铁，下了站万物俱籁，没有车，一个人打着伞在雪中走了回去。
后来就没有后来，我也忘了这个女孩的名字。

第二个女孩的故事是这样，某年我为某个实习生办理入职，后来发生了一些意外，几个月后，她意外被辞。
我说我喜欢玩游戏，可是我不能玩。我说我想买游戏机，可是我没有钱。
她说她也想买，可好玩了。
后来我还模糊记得这个女孩的脸，但是我也忘了她的名字。

今天希望把 第三章 目标文件看完。

————2020.12.22

今天又去看C ，C Primer plus 这本书确实需要好好看。它非常适合反复观看，并且事无巨细。
可能不适合我非常详细的去看，看起来内容不多，慢慢看就好了。
后来还是觉得直接上代码好了，不懂就查。
东西大概有个映像，不断的使用就能体会到一些东西。
为啥我本来是linux 0.12 ,跑去看 C 了呢，发现我貌似忘了 C 。
把哈工大的操作系统课程前3节课程再看了一遍，没有之前的觉得老师讲的垃圾的想法了。
确实讲到了重点。
看linux 完全注释这本书的时候，非常烦躁。
看的昏昏欲睡，该书的前5章都没有讲linux 0.12 的任何一句代码。
但实际上都和Linux 0.12 密切相关。
加上我本来加就看过保护模式的相关书籍，因此心态上不免有些无所谓。

————2020.12.23

今日摸鱼，看代码看不进去

————2020.12.25

今日取早上看了下杭市图书馆的公众号，发现之前想借的人别人已经还了，果断2块钱公交过去，借到手。

本来准备今天在杭市图书馆呆一天，但是太过于吵闹了，后面的小孩吵得看不进去。

打开包的瞬间，愣了，我说为什么觉得包这么轻，电脑没在包里。

早上约了宽带师傅，本来准备放他鸽子，正好回来。

好几天都没网，看猫光信号没有，猜测施工搞断了。

整幢楼别人都不用网了吧，本来我准备等着别人叫。

今天过来检查果然光纤又被挖断了。

这么折腾一通下来，下午了，于是又在家当了一天废物。

————2020.12.26

今日终于把 linux 0.12 的 bootsect 部分看完了，这篇文章的笔记在这里

————2020.12.27

今日图书馆关门，在家摸鱼。
晚上把 Linux 0.12 的 setup部分完成了，笔记已经大致写完。这部分内容不是很复杂，明天我分析完 head一并发布。

今天偶然看到一句“他就是越单身的人，越不容易找到另一半，你单身时间久了，这人就过独了”。
我已经习惯了很久了一个人生活，我很难想象当下有另一个人闯入我的生活，哪怕这个人我很喜欢她。
饮食、习惯、思想我觉得很难去主动适应另一个人，我也不大愿意被另外一个人说着，这样我会觉得非常别扭。
如果我未来真的有另一半，我觉得应该有一个试探这个边界的过程。

————2020.12.28

昨日下午在临时耽搁，去了一趟沪，参加某个大会，期间发生了一些事：

  • 去酒店的时候被个大叔错误指路，然后偶然走进员工通道，然后又被某个大叔指路某道门打开之后就是目标的会场，非常意外。

  • 旁边的小姑娘换了几批，最后一位姑娘听会过程中满不停的记笔记，仿佛是某个时候的我，当然现在我意识到这没有用。

  • 沪的冬天风大、有太阳，但还是很冷。

  • 这种大会少参加，非常无聊，能看直播就看直播好了。

  • 回杭的路上错误估计时间，改签一次之后仍然没赶上，车票作废，心疼我的钱。

  • 昨日杭雨夹雪，我没看见，在沪的我被雨夹雪伴随着大风打在脸上生疼。

  • 写这篇日常的时候，已经 2020 年最后一天的凌晨，2020年任然是个废物。
